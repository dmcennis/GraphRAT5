Graph-RAT 0.4 Users Guide

Graph-RAT is a powerful language for performing data analysis. Its speciality is mixing traditional analysis of social networks and traditional machine learning. As a result, there exists a number of algorithms implemented from sociology, graph theory, and machine learning. This guide will outline how to construct applications in this framework to solve real world problems.

First, one should read the overview of Graph-RAT to understand this rather unusual environment

Next, in constructing applications, there are three sets of choices to make: how to get the data, what type of graph object to use with the data, and what algorithms to execute over the data.

  • Getting Data: How to get the data into Graph-RAT for analysis.
  • Picking the graph type: Choose which graph type best suites the application purpose.
  • AlgorithmsOverview of what algorithms are and how to combine them.

For reference, all algorithm descriptions and a number of sample Graph-RAT programs are also provided.

  • AlgorithmGlossary: description of all algorithms in Graph-RAT by category.
  • Examples: example software using Graph-RAT.
