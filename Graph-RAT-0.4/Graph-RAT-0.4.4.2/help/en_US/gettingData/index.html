Getting Data

There are three different ways to get data into Graph-RAT for analysis. One is to use the WebCrawler. Another is to create an XML file. The final way is to construct a Graph-RAT database.
  • Webcrawling: CrawlLiveJournal provides a ready-made algorithm for this purpose, crawling the blog site LiveJournal for FOAF user descriptions.
  • Graph-RAT XML: Construct an xml file outside Graph-RAT conforming to the Graph-RAT's MemGraph's XML format for loading a graph.
  • Graph-RAT DB: Construct a database that conforms to DarbyDB's or PostgresqlDB's syntax, then load the graph using a custom graph type.
