Algorithm Glossary

There are many algorithms pre-defined in Graph-RAT. They fall into the categories:

  • Data Aquisition algorithms for collecting data.
  • Aggregation: Algorithms for aggregating data
  • Clustering: Algorithms for clustering data into subgraphs
  • Collaborative Filtering: Algorithms that use co-occurrence to generate links
  • Display: Algorithms for presenting Graph-RAT data
  • Evaluation: Algorithms for evaluating how well another algorithm has performed.
  • Machine Learning: Algorithms using Weka to create new links
  • Prestige: Algorithms that calculate the importance of a node
  • Similarity: Algorithms that calculate how alike nodes are

The details of specific algorithms are located in the javadoc under bibliography and more general algorithm headings.

There is also Transform to Array group.
